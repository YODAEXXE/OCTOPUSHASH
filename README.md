🔐 OCTOPUSHASH

A simple and educational Python hashing tool with a unique ASCII identity.

🧠 About the Project

OCTOPUSHASH is a command-line tool developed in Python that generates cryptographic hashes from user input.
It uses Python’s built-in hashlib library to demonstrate how different hashing algorithms work in practice.

The project focuses on learning core cybersecurity concepts such as hashing, data integrity, and algorithm strength.

⚙️ Features
🔑 Generate hashes from user input
🧾 Supports multiple hashing algorithms
📊 Basic security indicators for each algorithm
🖥️ Interactive CLI interface
🐙 Custom ASCII art branding
🔐 Supported Algorithms
Algorithm	Description
🔐 SHA-256	Secure and widely used hashing algorithm
🛡️ SHA-512	More robust with longer output
⚠️ MD5	Insecure and deprecated (for study purposes)
📊 SHA-224	Shorter variant of SHA-256
💪 SHA-384	Strong and balanced alternative
🧩 How It Works
📥 The user enters a password or text
📋 The program displays a menu of hashing algorithms
🎯 The user selects an option
⚡ The corresponding hash function is executed
🧾 The generated hash is displayed
🛠️ Technologies Used
🐍 Python 3
📦 hashlib (standard library)
🎯 Purpose

This project was created as a hands-on way to understand how hashing works and how different algorithms compare in terms of security.

🚀 Future Improvements
🔒 Add salting functionality
🗂️ Store and verify hashed passwords
🔓 Implement hash cracking (wordlist/brute force) for learning
🎨 Improve CLI design and user experience
