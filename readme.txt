This project is a Python-based hashing tool designed to generate and verify cryptographic hashes using multiple algorithms such as MD5, SHA-1, SHA-256, SHA-224, SHA-384, and SHA-512. It allows users to input a password or text and securely transform it into a hash representation.

The program includes a simple and interactive command-line interface, featuring an ASCII art octopus as its visual identity. Users can choose the hashing algorithm, generate hashes, and understand the relative security level of each method.

This project was developed as a learning exercise to explore fundamental concepts in cybersecurity, including hashing, data integrity, and basic password handling techniques.
